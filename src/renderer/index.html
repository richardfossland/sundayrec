🍎 macOS — aktivere oppvåkning fra hibernate
  1. Åpne Terminal (søk i Spotlight)
  2. Lim inn denne kommandoen og trykk Enter:
sudo pmset -a standbydelay 86400 && sudo pmset -a autopoweroff 0

Dette forhindrer Mac fra å slå seg helt av etter dvale. Skriv inn admin-passord når det spørres.

🪟 Windows — aktivere oppvåkning fra hibernate
  1. Åpne Kontrollpanel → Strømalternativer
  2. Klikk Endre planinnstillinger → Endre avanserte strøminnstillinger
  3. Under Dvale → sett Dvalemodus etter til Aldri (bruk kun Hvile)
  4. Alternativt: åpne PowerShell som administrator og kjør:
powercfg /hibernate off

Dette deaktiverer hibernate og beholder vanlig dvale (S3) som alltid kan vekkes.

⚠ ASIO på Windows

ASIO-modus fungerer ikke i SundayRec fordi programmet bruker nettleserens lyd-API (Web Audio / WASAPI). ASIO er et eget lydlag som bypasser Windows-mikseren fullstendig.

Har du et ASIO-kort? Bruk den gratis programvaren Voicemeeter Banana: still inn ASIO-kortet som inngang i Voicemeeter, rut lyden til en virtuell WASAPI-enhet, og velg den virtuelle enheten i SundayRec.

Nettverk-lyd (Dante / AES50 / AVB) støttes ikke direkte — bruk en Dante AVIO USB-adapter eller kortets innebygde USB-utgang.

Windows: velg WDM eller WASAPI i driverinnstillingene — ikke ASIO.
